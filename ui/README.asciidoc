== Setup 

=== Installation 

    forge --install ui 

=== As a dependency of another addon: 

Add this to your *pom.xml*: 

      <dependency>
         <groupId>org.jboss.forge.addon</groupId>
         <artifactId>ui</artifactId>
         <version>${forge.version}</version>
         <classifier>forge-addon</classifier>
      </dependency>
